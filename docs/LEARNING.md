# Git Learning Notes

## git init

Purpose:
Convert a normal folder into a Git repository.

## git add

Purpose:
Move files into staging area.

## git commit

Purpose:
Create a snapshot of project state.

## git push

Purpose:
Upload commits to GitHub.

## git pull

Purpose:
Download latest changes from GitHub.

## git clone

Purpose:
Download an existing repository.

## git remote

Purpose:
Connect local repository to GitHub.

## git branch

Purpose:
Manage branches.
